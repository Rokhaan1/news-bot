# X Account Growth Strategy

## The Account
- Established account, 2,000+ followers, paid X subscription (Premium).
- Goal: grow a personal following via timely, reliable news.

## Content Pillars (the 5 topics we post about)
1. **Global breaking news** — major world events.
2. **US foreign policy** — Washington's moves abroad (Iran, Venezuela, etc.).
3. **Afghanistan breaking news** — the home-base edge; cover fast and accurately.
4. **Football World Cup** — extremely timely (World Cup 2026 happening now).
5. **Afghanistan cricket** — passionate, underserved audience.

## Voice & Style
- Fast, factual, neutral headline-style. No hype, no "BREAKING" on unverified news.
- 1 clear fact per tweet. Add source/handle when possible.
- Use 1–2 relevant hashtags max. Emojis sparingly (🚨 for confirmed breaking).

## The Golden Rule
**Accuracy > speed.** One wrong "breaking" post damages trust permanently.
Always verify from a second source before posting hard news.

## Content Format Rules
- **Source credit:** every hard-news post cites the source — `(Source: Reuters)` or @handle. Non-negotiable for credibility.
- **Videos:** share as a **link** to the original (playable card) with our own caption + credit. Never re-upload someone else's video (copyright/credit).
- **Images/graphs:** auto-generate clean DATA graphics (standings tables, scorecards, timelines, bar charts) as PNGs via script — free. NO photo-like AI images (fake-news risk).

## Posting Cadence (start here)
- 3 news posts/day at fixed UTC slots, each timed to a target audience:
  - **14:00 UTC** → Afghanistan evening (~18:30) — lead with Afghan news.
  - **16:00 UTC** → UK late afternoon (~17:00) — World Cup / global.
  - **19:00 UTC** → US noon–afternoon (12:00 PT / 15:00 ET) — Afghanistan for the
    diaspora/policy audience (US-foreign-policy pillar underperformed at 0.0 avg).
- Weekly (Friday): a 3–5 tweet expert heritage deep-dive thread.
- Each slot falls back to the next-best pillar if its topic has no fresh news.
- Lead with whatever is genuinely breaking that day within the slot's audience.
- Reply to 5–10 bigger accounts in these topics daily (real engagement = growth).

## Growth Levers (the unsexy truth)
1. Consistency (daily, for months).
2. Being fast + reliable on Afghanistan/region news (your edge).
3. Genuine replies under big accounts in your niches.
4. Learn weekly: double down on posts that performed.

## Automation Mode: FULLY AUTOMATED (no manual posting)
The agent finds news, generates the graphic, writes the post, and publishes — hands-off.

### Automated safeguards (replace manual review)
- **2-source rule:** hard news posts only if confirmed by 2+ trusted sources; single-source = skip/hold.
- **Confidence gate:** low-risk items (sports scores, official statements) post freely; unconfirmed conflict claims auto-held.
- **Source credit:** auto-attached to every post.

## Roadmap
- Phase 1: One-time setup — X developer API keys (free tier) + cloud environment.
- Phase 2: Build the agent — monitor feeds, verify (2-source), generate graphic, post.
- Phase 3: Schedule it as a cloud routine — runs several times daily, no PC needed.
